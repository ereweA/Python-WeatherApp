# Python-WeatherApp

Weather API

Enter a location based on:


-City name (e.g., "London")


-City name and country (e.g., "London,UK")


-Postal or zip code (e.g., "94040")


-Latitude and longitude (e.g., "48.8566,2.3522")


-IP address (e.g., "192.168.0.1")


Returns:


-Feels Like


-Condition


-Windspeed


-Humidity


-Cloud Cover


-Visibility


Options:


-Switch between Miles/KM
